&#x20;**Superstore Sales Analysis Dashboard (Power BI)**



&#x20;**Project Overview**



&#x20;\* This project analyzes retail sales data using Microsoft Power BI to

&#x20;  uncover key business insights.



&#x20;\* The interactive dashboard highlights sales performance, profitability, customer behavior,

&#x20;  and regional trends, enabling data-driven decision-making..





&#x20;**Tools Used**



* Microsoft Power BI
* Power Query (Data Cleaning \& Transformation)
* DAX (Data Analysis Expressions)





📁 **Dataset**



&#x20;Superstore dataset including:



&#x20; \* Sales, Profit, Discount

&#x20; \* Customer Name, Segment

&#x20; \* Region, Category, Sub-Category

&#x20; \* Order Date and Ship Date





&#x20;📊 **Key Insights**



* The West region recorded the highest sales performance
* The Technology category generated the highest profit
* Higher discounts negatively impact profit margins
* A small group of top customers contributes significantly to revenue

&#x20;  Sales show a seasonal trend with peak performance in Q4)

🚀 **Features**



\* Interactive filters (Region, Segment, Category)

\* Sales and Profit analysis by Region and Category

\* Top 10 Customers by Sales

\* Clean and interactive dashboard design







📈 **Business Value**



* Identifies high-performing regions and product categories
* Provides insights into customer purchasing patterns
* Supports strategic decisions to improve profitability
* Helps track sales trends and seasonal demand
